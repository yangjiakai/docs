## 概述 / Overview

next-forge 在 `.github/dependabot.yml` 中配置了 Dependabot，每月检查更新。当有包更新时，将会打开一个拉取请求。

## 建议 / Recommendations

你可能想考虑使用像 Socket 这样的依赖项分析工具来检查拉取请求中的依赖项问题。我们还建议启用 [GitHub Secret Scanning](https://docs.github.com/en/code-security/secret-scanning/introduction/about-secret-scanning) 或使用 [Gitleaks](https://github.com/gitleaks/gitleaks) 或 [Trufflehog](https://github.com/trufflesecurity/trufflehog) 等工具来检查代码中的敏感信息。
