next-forge 通过 Vercel Analytics 提供网站分析功能 — 这是一个专注于网站性能和使用情况的分析平台。

## 使用方法

要启用网站分析，你需要在项目的根目录下配置 `next.config.js`：

```js
module.exports = {
  analytics: {
    vercel: {
      enabled: true,
    },
  },
};
```

## 可用指标

Vercel Analytics 提供以下关键指标：

- **网页浏览量** - 跟踪页面访问次数
- **独立访客** - 统计独立用户数量
- **地理分布** - 查看访客的地理位置分布
- **设备类型** - 了解访客使用的设备类型
- **性能指标** - 监控网站性能和加载时间

## 实时数据

你可以在 Vercel 仪表板中实时查看这些指标：

1. 登录到你的 Vercel 账户
2. 选择你的项目
3. 点击 "Analytics" 标签页
4. 查看实时和历史数据

## 自定义事件

除了默认的页面浏览跟踪外，你还可以添加自定义事件跟踪：

```tsx
import { track } from '@vercel/analytics';

// 跟踪自定义事件
track('Button Clicked', { buttonId: 'signup' });
```

## 数据导出

你可以通过 Vercel Analytics API 导出原始数据进行深入分析：

```bash
curl -X GET "https://api.vercel.com/v1/analytics/events" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
