title: 调试
description: 如何调试你的应用程序
icon: bug
---

next-forge 已经预先配置了 [VS Code](https://code.visualstudio.com/) 来作为调试器。`.vscode/launch.json` 文件包含了调试器的配置，并且已经配置好了所有应用程序的调试。

要使用调试器，只需要在 VS Code 中打开应用程序（或任何兼容的编辑器），然后进入调试面板（Windows/Linux：Ctrl+Shift+D，macOS：⇧+⌘+D）。选择一个启动配置，然后按 `F5` 或从命令面板中选择 `调试：开始调试` 来启动调试会话。
