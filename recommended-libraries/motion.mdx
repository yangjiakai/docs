## 什么是 Motion One？ / What is Motion One？

[Motion One](https://motion.dev/) 是一个现代的网页动画库，它提供了高性能、轻量级的动画解决方案。它是 GSAP 的一个很好的替代品。

## 为什么使用它？ / Why Use It？

Motion One 提供了许多优势：

- 极小的包体积（仅 4.5kb gzipped）
- 使用 Web Animations API
- 流畅的性能
- 简单的 API
- TypeScript 支持

## 安装 / Installation

使用以下命令安装：

```bash
pnpm add @motionone/dom
```

## 基本用法 / Basic Usage

### 简单动画 / Simple Animation

```typescript
import { animate } from "@motionone/dom"

// 简单的淡入效果
animate("#element", { opacity: 1 })

// 带有配置的动画
animate("#element", 
  { 
    opacity: 1,
    transform: "translateY(0px)" 
  },
  { 
    duration: 1,
    easing: "ease-out" 
  }
)
```

### 时间轴动画 / Timeline Animation

```typescript
import { timeline } from "@motionone/dom"

const sequence = [
  [
    "#element-1",
    { opacity: 1, transform: "translateY(0px)" }
  ],
  [
    "#element-2",
    { opacity: 1, transform: "scale(1)" },
    { at: "-0.5" }
  ]
]

timeline(sequence)
```

## 高级特性 / Advanced Features

- 支持 CSS 变量
- 支持 Spring 动画
- 支持手势动画
- 支持滚动动画

## 性能优化 / Performance Optimization

Motion One 自动使用硬件加速来优化动画性能。它还提供了一些优化建议：

- 优先使用 `transform` 和 `opacity` 属性
- 避免同时动画过多元素
- 使用 `will-change` 提示浏览器
