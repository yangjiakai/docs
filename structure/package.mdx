---
title: 依赖管理
description: next-forge 的依赖管理,
icon: cube
---

next-forge 包含多个在整个 monorepo 中共享的包。这些包的目的是将共享代码与主应用隔离，使其更容易管理和更新。

此外，这也使得替换应用的不同实现部分变得更容易。例如，`database` 包包含了所有与数据库相关的内容，包括数据库模式和迁移。这使我们能够在不影响应用其他部分的情况下轻松更换数据库提供商或 ORM。

| 包名 | 描述 |
| ------- | ----------- |
| `analytics` | 分析系统 |
| `auth` | 身份认证提供者 |
| `database` | 数据库模式和迁移 |
| `design-system` | 包含共享组件、实用工具文件和样式 |
| `email` | 电子邮件功能，包括提供商 SDK 和模板 |
| `env` | 环境变量 |
| `feature-flags` | 功能标志 |
| `next-config` | 共享的 Next.js 配置 |
| `observability` | 可观测性工具，包括日志记录、错误处理和正常运行时间监控 |
| `payments` | 支付和计费功能 |
| `seo` | 搜索引擎优化，包括元数据和结构化数据 |
| `tailwind-config` | 共享的 Tailwind CSS 配置 |
| `typescript-config` | 共享的 TypeScript 配置 |
