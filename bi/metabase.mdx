虽然 next-forge 默认不包含商业智能工具，但你可以轻松地使用 [Metabase](https://www.metabase.com/) 为你的应用添加商业智能和分析功能。

## 概述 / Overview

Metabase 是一个开源的商业智能平台。你可以使用 Metabase 查询数据，或将其嵌入到你的应用中，让客户能够自主探索他们的数据。

## 安装 Metabase / Installing Metabase

Metabase 通过 Dockerhub 提供官方 Docker 镜像，可用于在任何运行 Docker 的系统上部署。以下是启动运行 Metabase 容器的一行命令：

```sh
docker run -d --name metabase -p 3000:3000 metabase/metabase
```

你也可以通过运行 JAR 文件来安装 Metabase。不同安装方式的链接：

- [快速开始](https://www.metabase.com/start/oss/)
- [Docker 文档](https://www.metabase.com/docs/latest/installation-and-operation/running-metabase-on-docker)
- [Jar 文件文档](https://www.metabase.com/docs/latest/installation-and-operation/running-the-metabase-jar-file)

## 数据库连接 / Database Connection

默认情况下，next-forge 使用 Neon 作为数据库提供商。Metabase 与 Postgres 数据库配合得很好，如果你准备连接，以下是你需要的信息：

- 数据库服务器的 `hostname`（主机名）
- 数据库服务器使用的 `port`（端口）
- `database name`（数据库名称）
- 你使用的数据库 `username`（用户名）
- 你使用的数据库 `password`（密码）

你可以在 `DATABASE_URL` 环境变量中找到这些详细信息：

```js
DATABASE_URL="postgresql://[username]:[password]@[hostname]:[port]/[database_name]?sslmode=require"
```
