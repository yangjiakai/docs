next-forge 使用 [Mintlify](https://mintlify.com) 来生成精美的文档。每个页面都是一个 `.mdx` 文件，使用 Markdown 编写，内置了 UI 组件和 API 交互功能。

## 创建新页面 / Creating a New Page

要创建新的文档页面，只需在 `apps/docs` 目录中添加一个新的 MDX 文件。文件名将用作页面的 slug，frontmatter 将用于生成文档页面。例如：

```mdx apps/docs/hello-world.mdx
---
title: '快速开始 / Quickstart'
description: '在五分钟内开始构建现代文档 / Start building modern documentation in under five minutes.'
---
```

了解更多支持的[元标签 / Meta Tags](https://mintlify.com/docs/page)。

## 添加页面到导航 / Adding a Page to Navigation

要将页面添加到侧边栏，你需要在 `apps/docs` 目录的 `mint.json` 文件中定义它。基于前面的示例，以下是如何将其添加到侧边栏：

```mdx mint.json {2-5}
"navigation": [
  {
    "group": "入门指南 / Getting Started",
    "pages": ["hello-world"]
  },
  {
    // ...
  }
]
```
