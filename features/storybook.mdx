next-forge 使用 [Storybook](https://storybook.js.org/) 作为设计系统的前端工作坊。它允许你与设计系统中的组件进行交互，并查看它们在不同状态下的表现。

## 配置 / Configuration

默认情况下，Storybook 配置了来自 [shadcn/ui](https://ui.shadcn.com/) 的所有组件，并允许你与它们交互。它还配置了相关的字体和高阶组件，以确保你的应用程序和 Storybook 之间的体验一致。

## 运行工作坊 / Running the Workshop

当你运行 `pnpm dev` 时，Storybook 将自动启动。你也可以使用 `pnpm dev --filter storybook` 独立启动它。预览将在 [localhost:6006](http://localhost:6006) 上可用。

## 添加故事 / Adding Stories

你可以通过在 `apps/storybook/stories` 目录中添加组件来将自己的组件添加到工作坊中。每个组件应该有自己的 `.stories.tsx` 文件。
