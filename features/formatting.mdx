next-forge 使用 [Ultracite](https://ultracite.dev) 进行代码格式化和代码检查。Ultracite 是一个预配置的 [Biome](https://biomejs.dev) 设置，Biome 是一个基于 Rust 的高性能工具链，包括格式化器、代码检查器等，用于 JavaScript 和 TypeScript。

## 优势 / Benefits

Ultracite 提供了以下优势：

- <Icon icon="check" iconType="solid" /> 零配置 - 开箱即用 / Zero configuration required - works out of the box
- <Icon icon="check" iconType="solid" /> 极快的性能 / Extremely fast performance
- <Icon icon="check" iconType="solid" /> 项目中的一致代码风格 / Consistent code style across your project
- <Icon icon="check" iconType="solid" /> 内置代码检查规则 / Built-in linting rules
- <Icon icon="check" iconType="solid" /> TypeScript 支持 / TypeScript support

## 使用方法 / Usage

格式化器和代码检查器在你的项目中自动配置，并将在保存时运行。如果你想手动在项目中运行它们，可以运行以下命令：

```sh Terminal
npx ultracite
```

这将检查并修复当前目录中的所有文件。
