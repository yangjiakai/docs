## 默认配置 / Default Configuration

next-forge 有一个专门的 JSON+LD 助手，用于创建经过完全验证的 Google 结构化数据，使你的内容更有可能在 Google 搜索结果中被突出显示。

默认情况下，结构化数据在以下页面上实现：

- 博客索引页面使用 `Blog` 类型
- 博客文章页面使用 `BlogPosting` 类型

## 使用方法 / Usage

我们的 `@repo/seo` 包提供了一个基于 `schema-dts` 构建的 JSON+LD 助手，允许以类型安全的方式生成结构化数据。你可以像这样声明自己的 JSON+LD 实现：

```page.tsx
import { JsonLd } from '@repo/seo/json-ld';
import type { WithContext, YourInterface } from '@repo/seo/json-ld';

const jsonLd: WithContext<YourInterface> = {
  // ...
};

return <JsonLd code={jsonLd} />;
```
